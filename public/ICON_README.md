# PWA Icon Generation

To generate proper PWA icons, you can:

1. Use an online tool like https://realfavicongenerator.net/
2. Upload the icon.svg file from this directory
3. Download the generated icons and replace icon-192.png and icon-512.png

Or use this simple approach:
- Open icon.svg in a browser
- Take a screenshot or use browser dev tools to export as PNG
- Resize to 192x192 and 512x512

For now, placeholder images are used. Replace them with actual PNG files for production.
